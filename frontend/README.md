# 3Botlogin frontend

The frontend for 3Bot login

## Project setup

```bash
npm install
```

### Configuration

```bash
make prod/staging/testing
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
